# Laravel 5.1 helper
### It reads the database table, then it generates the following codes / files:

* Routes
* RouteServiceProvider
* Some artisan commands
* Nav menu
* Model
* Request
* Controller
* Views [create, edit, index, show] based on Bootstrap
